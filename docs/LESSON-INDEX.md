## N4 阶段（剩余 10 课）

### 第 11 課 · て形連結 ✏️
- **語法**：〜て、〜（动作连接）/ 〜てから / 〜ています（继续）
- **主篇**：朝のルーティン（200-300 字描述早晨日常）
- **配篇**：引っ越しの日（搬家当天的事件序列）
- **对话**：同事询问昨天做了什么，需用〜て、〜串联
- **重点**：て形是 N4 第一大难关，本课要把动词分类（I/II/III）的て形变化讲透

### 第 12 課 · 普通形入門 ✏️
- **語法**：常体（动词・形容词・名词的普通形）/ 常体的会话感 / 表内交互列表
- **主篇**：友達とのチャット（朋友间的 LINE 聊天记录形式）
- **配篇**：日記の一日（用常体写的日记）
- **对话**：前辈向后辈解释什么场合用常体
- **重点**：常体不是"不礼貌"，而是亲密度的标志

### 第 13 課 · 可能形 ✏️
- **語法**：〜ことができます / 可能动词（行ける・食べられる）/ 〜が話せます
- **主篇**：私のできること（自我介绍能力）
- **配篇**：外国語の学習（学外语的能力进阶）
- **对话**：面试场景，询问求职者能做什么
- **重点**：可能动词的活用规则；可能形助词从「を」变「が」

### 第 14 課 · たい・ほしい ✏️
- **語法**：〜たい / 〜がほしい / 第三者的愿望（〜たがる）
- **主篇**：将来の夢（写自己的梦想）
- **配篇**：誕生日プレゼント（朋友想要什么生日礼物）
- **对话**：朋友间讨论假期想做什么
- **重点**：〜たい和〜がほしい的区别；不能直接说别人的愿望

### 第 15 課 · 比較 ✏️
- **語法**：〜より / 〜のほうが / 〜の中で〜が一番 / どちら
- **主篇**：日本と中国（中日文化比较）
- **配篇**：四季の好み（讨论喜欢哪个季节）
- **对话**：朋友间比较两个城市的优劣
- **重点**：「より」和「ほうが」的搭配；最高级用「中で」

### 第 16 課 · 試しと経験 ✏️
- **語法**：〜てみる（尝试）/ 〜たことがある（经验）
- **主篇**：初めての温泉（第一次去温泉的经历）
- **配篇**：面白い経験（人生中有趣的经历）
- **对话**：朋友推荐尝试新事物
- **重点**：「ことがある」表经验，注意「ない」的过去形「なかったことがある」

### 第 17 課 · あげる・もらう・くれる ✏️
- **語法**：あげる（我/第三方→他人）/ もらう（他人→我）/ くれる（他人→我）
- **主篇**：お土産（旅游回来送特产给同事）
- **配篇**：家族からの手紙（来自家人的信和礼物）
- **对话**：感谢朋友送的礼物
- **重点**：日语授受动词的视角概念，是日本文化中「内」「外」的体现

### 第 18 課 · 条件 ① ✏️
- **語法**：〜たら（如果...就...）/ 〜と（一...就...，自然结果）
- **主篇**：天気が良かったら（假设天气好的安排）
- **配篇**：このボタンを押すと（机器使用说明）
- **对话**：商量周末计划，"如果天气好就..."
- **重点**：「たら」是最常用的条件，「と」用于必然性结果（自然现象、机器操作）

### 第 19 課 · 条件 ② ✏️
- **語法**：〜ば（一般条件，如果...就...）/ 〜なら（接对方话题，"如果是...的话"）
- **主篇**：もし時間があれば（如果有时间想做的事）
- **配篇**：東京へ行くなら（去东京的话推荐）
- **对话**：朋友咨询旅游建议，用「〜なら」回应
- **重点**：4 种条件（たら・と・ば・なら）的微妙差异；「なら」的话题接收特点

### 第 20 課 · 引用と思考 ✏️
- **語法**：〜と言いました（引用）/ 〜と思います（认为）/ 〜そうです（伝聞，听说）
- **主篇**：ニュースで聞いた話（新闻里听到的事）
- **配篇**：友達の意見（朋友的意见）
- **对话**：转述别人的话，发表自己看法
- **重点**：直接引用 vs 间接引用；「思う」前用普通形

## N3 阶段（18 课）

### 第 21 課 · 受身（被动）✏️
- **語法**：动词被动形 / 被动的迷惑用法（被害感）/ 物品被动
- **主篇**：雨に降られた日（被雨淋的一天，迷惑被动）
- **配篇**：先生にほめられました（被老师表扬）
- **对话**：抱怨地铁里被人撞、东西被弄坏
- **重点**：日语被动除"被某人做"还有"受到困扰"的含义

### 第 22 課 · 使役 ✏️
- **語法**：〜させる（使...）/ 〜させてください（请允许我...）
- **主篇**：部下に書かせる（让下属写报告）
- **配篇**：子供を留学させる（让孩子去留学）
- **对话**：上司给下属分配工作；学生请求老师允许做某事
- **重点**：使役有"强制让做"和"允许做"两种语感

### 第 23 課 · 使役受身 ✏️
- **語法**：〜させられる / 区别动词 I 类的「飲まされる」缩约形
- **主篇**：飲み会の思い出（被前辈强行劝酒的回忆）
- **配篇**：学生時代（学生时代被强迫做不愿做的事）
- **对话**：朋友间吐槽职场被迫加班、聚会
- **重点**：使役受身表达"被迫做不愿做的事"；动词 I 类有缩约形

### 第 24 課 · 敬語入門 ① 尊敬語 ✏️
- **語法**：尊敬动词（いらっしゃる・召し上がる・なさる など）/ お〜になる / 〜れる/られる（尊敬）
- **主篇**：社長のスピーチ（社长讲话的转述）
- **配篇**：お客様への対応（接待客人的标准用语）
- **对话**：店员对顾客；秘书向上司汇报
- **重点**：日语三大敬语体系的概览；尊敬语的核心是抬高对方

### 第 25 課 · 敬語入門 ② 謙譲語 ✏️
- **語法**：谦让动词（参る・申す・いたす など）/ お〜する / 〜させていただく
- **主篇**：面接で（面试中的自我介绍）
- **配篇**：メールの書き方（商务邮件标准格式）
- **对话**：求职者面试；下级向客户汇报
- **重点**：谦让语的核心是降低自己；区分丁宁语 vs 谦让语

### 第 26 課 · 様態（外観・伝聞・推量）✏️
- **語法**：〜そう（看起来）/ 〜らしい（听说/像）/ 〜みたい（像/似乎）
- **主篇**：あの店、おいしそう（看到的店铺看起来好吃）
- **配篇**：彼は忙しいらしい（听说他很忙）
- **对话**：朋友间分享小道消息和外观判断
- **重点**：三个近义表达的微妙差异；〜そう接续规则

### 第 27 課 · 必要と義務 ✏️
- **語法**：〜なければならない / 〜べき / 〜ことになっている
- **主篇**：会社のルール（公司规定）
- **配篇**：日本の常識（在日本必须遵守的常识）
- **对话**：新员工询问公司规则
- **重点**：三个近义语法的强弱程度；「べき」带主观判断

### 第 28 課 · 許可と禁止 ✏️
- **語法**：〜てもいい / 〜てはいけない / 〜ないでください / 〜ても構わない
- **主篇**：図書館で（图书馆里的注意事项）
- **配篇**：マンションのルール（公寓规则）
- **对话**：客人询问主人某事是否可以做
- **重点**：礼貌程度的梯度；〜なくてもいい（不必）vs 〜てはいけない（禁止）

### 第 29 課 · 推量 ✏️
- **語法**：〜だろう / 〜かもしれない / 〜にちがいない / 〜と思われる
- **主篇**：明日の天気（天气预报式推测）
- **配篇**：あの人は誰だろう（推测某人身份）
- **对话**：朋友间猜测公司新人的背景
- **重点**：四个推量表达的确定度从低到高排序

### 第 30 課 · 接続表現 ① ✏️
- **語法**：〜のに / 〜ても / 〜けれど / 〜にもかかわらず（高级）
- **主篇**：努力したのに（明明努力了却没结果）
- **配篇**：雨でも行きます（即使下雨也要去）
- **对话**：抱怨意外结果
- **重点**：「のに」带强烈感情色彩（不满、惋惜）；「ても」中性

### 第 31 課 · 接続表現 ② ✏️
- **語法**：〜ながら（同时）/ 〜たまま（保持状态）/ 〜ついでに（顺便）
- **主篇**：散歩しながら（边散步边思考）
- **配篇**：買い物のついでに（顺便办事）
- **对话**：描述自己的多任务习惯
- **重点**：「ながら」需主语相同；「たまま」表示状态延续

### 第 32 課 · 時を表す表現 ✏️
- **語法**：〜うちに / 〜間に / 〜とき / 〜ところ（直前/现在/直后）
- **主篇**：若いうちに（趁年轻该做的事）
- **配篇**：ちょうど食べているところ（正吃饭时电话来了）
- **对话**：朋友约见面，用「〜ているところ」描述当前状态
- **重点**：「ところ」三个时态的精确含义；「うちに」vs「間に」

### 第 33 課 · 目的 ✏️
- **語法**：〜ために（目的，主语相同）/ 〜ように（结果，常用于无意志动词）
- **主篇**：健康のために（为了健康做的事）
- **配篇**：忘れないように（为了不忘记的小窍门）
- **对话**：讨论学外语的方法
- **重点**：「ために」前接意志动词，「ように」前接非意志动词或可能形

### 第 34 課 · 結果と変化 ✏️
- **語法**：〜ようになる（自然变化）/ 〜ようにする（自我决定）/ 〜ことにする / 〜ことになる
- **主篇**：日本語が話せるようになりました（学会了说日语）
- **配篇**：会社を辞めることにしました（决定辞职）
- **对话**：朋友报告人生重大决定
- **重点**：「ことにする」（自己决定）vs「ことになる」（决定结果）

### 第 35 課 · 程度と限定 ✏️
- **語法**：〜ほど / 〜だけ / 〜しか / 〜ばかり
- **主篇**：泣くほど嬉しかった（高兴得想哭的程度）
- **配篇**：甘いものばかり食べる（光吃甜食的负面评价）
- **对话**：朋友间互相吐槽生活习惯
- **重点**：「だけ」中性 vs「しか〜ない」消极；「ばかり」常带负面语气

### 第 36 課 · 列挙と対比 ✏️
- **語法**：〜たり〜たり / 〜とか / 〜一方で / 〜に対して
- **主篇**：休みの日は（描述休息日的多种活动）
- **配篇**：都会と田舎（比较都市与农村）
- **对话**：朋友讨论各自周末做什么
- **重点**：「たり」表示举例（暗示还有其他）；「一方で」用于对比两个事实

### 第 37 課 · 慣用表現（推測の確信度）✏️
- **語法**：〜わけだ / 〜わけではない / 〜はずだ / 〜はずがない
- **主篇**：彼が来ないわけ（他不来的原因解释）
- **配篇**：そんなはずはない（不可能这样的强烈否定）
- **对话**：朋友间分析某个状况的原因
- **重点**：「わけ」表事理推论；「はず」表期待预期

### 第 38 課 · 感情と評価 ✏️
- **語法**：〜てしまう（遗憾完成）/ 〜ておく（事先准备）/ 〜ちゃう（口语缩约）
- **主篇**：忘れてしまった（一连串遗憾的事）
- **配篇**：準備しておこう（出门前的准备清单）
- **对话**：朋友间分享倒霉日；准备旅行
- **重点**：「てしまう」的两种语感（完成 / 遗憾）；「〜ちゃった」是常体口语版

## N2 阶段（12 课）

### 第 39 課 · 書き言葉入門 ✏️
- **語法**：だ・である体 / 「〜のだ」的硬性用法 / 主旨段
- **主篇**：现代社会について（500-600 字论说文，**首次使用 だ・である 体**）
- **配篇**：新聞のコラム（模仿新闻短评的风格）
- **对话**：编辑指导记者写文章（仍用礼貌体）
- **重点**：从口语到书面语的语气转换；如何写一段标准的"問題提起"

### 第 40 課 · 高級接続詞 ✏️
- **語法**：したがって / ところが / それゆえ / なお / ただし / さらに / 一方
- **主篇**：論文の書き方（论文写作的接续词使用指南）
- **配篇**：契約書の一節（合同条款的语气）
- **对话**：导师指导学生论文
- **重点**：书面接续词的精确用法；「ところが」与「しかし」的微差

### 第 41 課 · 観点と立場 ✏️
- **語法**：〜にとって / 〜について / 〜に関して / 〜をめぐって
- **主篇**：環境問題について（环境问题论说）
- **配篇**：教育をめぐる議論（围绕教育的辩论）
- **对话**：电视辩论节目片段
- **重点**：四个介词性短语的细微差别；「をめぐって」专用于争议话题

### 第 42 課 · 根拠と論理 ✏️
- **語法**：〜に基づいて / 〜を通じて / 〜からして / 〜ことから
- **主篇**：データに基づく分析（基于数据的分析报告）
- **配篇**：経験を通じて学んだこと（通过经验学到的事）
- **对话**：研究者向同行汇报研究方法
- **重点**：论文常用表达；如何陈述论据来源

### 第 43 課 · 強調と限定 ✏️
- **語法**：〜こそ / 〜さえ / 〜すら / 〜にすぎない
- **主篇**：今こそ行動を（呼吁行动的演讲文）
- **配篇**：ほんの始まりにすぎない（说"这只是开始"的谦逊语气）
- **对话**：演讲者激励听众
- **重点**：「こそ」强调句；「さえ」「すら」的语感强度

### 第 44 課 · 譲歩と逆接 ✏️
- **語法**：〜にもかかわらず / 〜とはいえ / 〜ものの / 〜ながらも
- **主篇**：努力したにもかかわらず（虽努力却失败的反思文）
- **配篇**：便利とはいえ（"虽方便但..."的批判式短文）
- **对话**：评论家对某社会现象的辩证看法
- **重点**：四个让步表达的书面感强弱；「ものの」常用于自我反省

### 第 45 課 · 抽象的な時間 ✏️
- **語法**：〜に際して / 〜に先立って / 〜次第 / 〜にあたって
- **主篇**：新年度に際して（新学年/财年开始的致辞）
- **配篇**：結果が出次第（公司通知"结果一出来就..."）
- **对话**：会议主持人开场白；秘书通知客户
- **重点**：仪式感、商务感强烈的时间表达

### 第 46 課 · 因果関係 ✏️
- **語法**：〜ばかりに / 〜あまり / 〜ことから / 〜だけに
- **主篇**：油断したばかりに（因疏忽导致严重后果的警示文）
- **配篇**：歴史的な経緯（从历史角度解释某现象的缘由）
- **对话**：长辈给晚辈讲教训
- **重点**：四个因果表达的语感（懊悔、过度、由此可见、正因为）

### 第 47 課 · 主観的判断 ✏️
- **語法**：〜ものだ（普遍真理 / 怀念）/ 〜ことだ（建议）/ 〜まい（决意/否定推量）
- **主篇**：人生というものは（关于人生的散文）
- **配篇**：二度と忘れまい（决心永不忘记的誓言）
- **对话**：长辈对年轻人讲人生道理
- **重点**：「ものだ」的多义性；「まい」是否定意志/推量的硬性表达

### 第 48 課 · 文化と社会 ✏️
- **語法**：综合应用前述书面体语法
- **主篇**：日本の四季と文学（800 字综合性散文，深度运用 N5-N2 所有语法）
- **配篇**：茶道の精神（茶道文化的内涵）
- **对话**：文化研究者讨论日本美学
- **重点**：综合应用，语言密度和深度同步上升

### 第 49 課 · 議論文の構造 ✏️
- **語法**：综合应用 + 论证、举例、归纳的句式（〜たとえば / つまり / 要するに / 言い換えれば）
- **主篇**：AIと人間の未来（关于 AI 与人类未来的议论文，有清晰的"論点 → 論拠 → 例 → 結論"结构）
- **配篇**：読書の意義（论述阅读的价值）
